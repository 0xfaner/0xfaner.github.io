简介

Hexo 是基于 Node.js 的一款静态博客框架，可以方便地生成静态网页托管在 GitHub 或服务器上。搭建博客的性价比之选，依赖少易于安装。

首先介绍下 Hexo 的机制：本地环境生成静态文件，将这些文件托管在平台上，这样用户只能访问到你生成的静态文件。所以评论这样的功能 Hexo 必须依靠第三方插件实现。但是好处是降低平台负担，仅需要托管即可。

本次部署环境：
